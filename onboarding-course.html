Flagship eLearning Case Study

Onboarding Course - New Employee Orientation

A 25-minute module designed to reduce time-to-productivity for new hires through branching scenarios, immediate feedback, and visual microlearning.

Tools: Articulate Storyline, Canva Audience: New employees Format: Self-paced

Project Snapshot

Quick framing of the case study: the core learner problem, the design intent, and the measurable outcomes this module targets.

Problem

New hires needed faster role clarity and decision confidence in their first week.

Design Goal

Build an experience that teaches policy through realistic choices and short reflection loops.

Learning Benefits

  • Faster onboarding consistency
  • Higher confidence in key tasks
  • Better knowledge retention

Storyline Learning System

This section shows how Storyline variables, branching logic, and immediate coaching feedback were combined to guide learner decisions.

This project now focuses on Storyline as the primary delivery mode: branching logic, variable-driven feedback, checkpoint progression, and learner coaching in a single interactive flow.

Interaction Engine (Storyline)

  • Branching path decisions that mirror day-one workplace choices
  • Triggers and variables that adapt feedback by learner action
  • Checkpoint knowledge checks with short, corrective remediation

Storyline Build Breakdown

Workflow
  1. SME interview and scenario mapping
  2. Storyboard drafting with feedback loops
  3. Branch logic + variable implementation
  4. QA pass, accessibility check, and publish
Interaction Components
  • Scene-based decisions with consequence text
  • Immediate corrective coaching after risky actions
  • Progress checkpoints before final outcome
  • Replay option for mastery reinforcement
Outcome Targets
  • Reduce onboarding confusion in first week
  • Increase policy-compliant decision confidence
  • Shorten time-to-productivity for new hires
